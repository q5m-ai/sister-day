# Sister Day

The mirror date across winter's coldest point.

**Sister Day** — For a given location, let *C* denote the calendar date with the lowest long-term average daily temperature. The *sister day* of a date *D* is the unique date *D'* equidistant from *C* on the opposite side of the annual temperature cycle — if *D* falls *n* days after *C*, then *D'* falls *n* days before *C*, and vice versa.

## Live

[sisterday.app](https://q5m-ai.github.io/sister-day)

## Features

- Calculate sister days for any date across 40+ cities worldwide
- Orbital SVG visualization of the annual temperature cycle
- Thermal lag explainer — why the coldest day isn't the winter solstice
- Fully responsive, single-page app with zero dependencies
